# Toast

Componente desenvolvido para exibir notificações.

### Como usar:

1.Faça sua importação

```js
import { toast } from '~/components/CustomToast';
```

2.Chame a função para abrir o toast de acordo com o tipo que deseja.

```js
toast.default('Default', 'Descrição Default');
toast.success('Sucesso', 'Descrição do sucesso');
toast.error('Erro', 'Descrição do Erro');
toast.warn('Warn', 'Descrição do Warn');

// Incluindo botão e link para redirecionamento
toast.success('Mensagem de sucesso', '', 'configurar', 'https://google.com');
```
